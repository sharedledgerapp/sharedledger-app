- Mobile-first design approach strictly followed
- Auth flow handles both "Create Family" and "Join Family"
- "Numeric" types from DB are strings in JSON, coerced to numbers for charts
- Receipt uploads use standard FormData to /api/upload
- Icons via Lucide React
